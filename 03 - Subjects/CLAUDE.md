# 03 - Subjects — Folder Guide

This folder contains **7 subject container notes**. Subjects are cross-cutting lenses — they slice across categories to group notes by theme.

---

## Subjects vs Categories

- **Category** = what type of thing the note is (a permanent note, a newsletter, an SOP...)
- **Subject** = what topic the note is about (business, creativity, productivity...)

A note can belong to one category but multiple subjects. Example: a permanent note about "Writing as thinking" could have `subjects: [[Creativity]], [[Productivity]]`.

---

## How to Use

Add subjects to any note:

```yaml
subjects:
  - "[[Creativity]]"
  - "[[Productivity]]"
```

The subject container note will automatically surface that note via its Bases query.

---

## Rules

- Never add notes directly to this folder
- A note doesn't need a subject — it's optional
- Add multiple subjects freely — subjects are lenses, not buckets
