# Horus Heresy Project Inspiration

My main hobby focus is the Horus Heresy. Set 10,000 years before Warhammer 40k, it's probably the richest period in the Warhammer universe for backstory — with over seventy novels and source books exploring the events of the Warmaster's betrayal of the Emperor of Mankind.

This is an evolving index of hobby project ideas. Each bullet will become a linked note as the idea develops into a project. New ideas land here first as bullets; the note becomes a MOC as those links are created.

> **To link a project note back to this MOC:** add `[[Horus Heresy Project Inspiration]]` anywhere in the project note body and it will appear in the Project Notes index below automatically.
