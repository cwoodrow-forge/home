# Zettelkasten Method SOP

**Goal:** Transform content highlights into a connected network of atomic ideas that generate original thinking and fuel content creation.

### Workflow Summary
1. **Literature Notes** — Summarize highlights from sources in your own words
2. **Permanent Notes** — Extract one atomic idea per note
3. **Connect Notes** — Link notes to build a knowledge network

---

---

### Step 1: Create Literature Notes

1. Open a literature note you've captured highlights in (or create one using the **Literature Note Template**)
2. Open the source material in a side pane
3. Work through your highlights one by one — rewrite each in your own words in the **Summary** section
4. One summary section per concept. If a highlight contains two ideas, write two summaries.
5. Set `status: complete` when done

> The goal isn't to summarize the whole source — it's to extract the ideas worth keeping. Follow your curiosity.

---

### Step 2: Create Permanent Notes

From your literature note summaries, extract atomic ideas — one concept per note.

**The atomicity rule:** One note = one idea. If the title needs "and", it's two notes.

1. Create a new note using the **Permanent Note Template** — title it with the idea itself (e.g., *"Consistency Beats Intensity in Skill Building"*)
2. Write the **Big Idea** in 2–4 sentences, in your own words
3. Add wikilinks to related notes in the **Links** section
4. Set `status: seedling`

---

### Step 3: Connect Notes

After writing a permanent note, use these trigger questions to find connections:

- How does this fit into what I already know?
- Could this idea explain or extend another note in my vault?
- What does this idea contradict?
- Why am I drawn to this idea?
- What are 3 ways I could apply this concept?

Add wikilinks (`[[Note Title]]`) wherever connections feel natural. One genuine connection beats ten forced ones.

---

### Build the Habit

**Daily:** Drop captures and rough ideas into `00 - Inbox/`
**Weekly:** Process 1–3 literature notes, write 2–5 permanent notes, review connections
