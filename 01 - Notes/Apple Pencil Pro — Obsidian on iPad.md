# Apple Pencil Pro — Obsidian on iPad

> Practice note. Write directly on this page in Obsidian edit mode. The Practice Area at the bottom is yours to scribble on.

---

## Before You Write — The Two Rules

**1. Always be in Edit mode.**
Tap the pencil icon (toolbar, top right) or swipe up from the bottom edge of the note. If you're in Reading mode, Scribble has nothing to write into.

**2. Tap to place your cursor first.**
Scribble inserts text at the cursor position — not where your pencil physically touches. Tap the exact word gap or line where you want to write, see the cursor blink, then start writing.

---

## Scribble Gesture Reference

| Gesture | How to do it | What it does |
|---|---|---|
| **Write normally** | Print or cursive anywhere in the text area | Converts to typed text at cursor |
| **Delete a word** | Scribble back and forth over it (scratch out) | Removes the word |
| **Insert a word** | Draw a ^ (caret) between two words, then write above it | Opens a gap and inserts text |
| **Join two lines** | Draw a vertical line connecting the end of one line to the start of the next | Merges them into one line |
| **Select text** | Draw a circle around words | Highlights the selection |
| **Add a space** | Draw a short horizontal line between two run-together words | Inserts a space |
| **New paragraph** | Write a vertical line at the end of a sentence, then write below | Starts a new paragraph |

## Your Three Specific Problems

### Problem 1 — Punctuation not recognised

Scribble sometimes misses punctuation because it's written too small or too quickly.

**Fixes:**
- Write punctuation **slightly larger** than you normally would — a comma the size of a letter c, a period as a deliberate dot
- **Pause briefly before and after** punctuation marks — a half-second gap signals "this is a symbol, not a letter"
- For a **full stop**: make a firm, intentional dot — don't trail off from the last letter into a dot
- For a **comma**: write a clear small c-shape with a hook — don't rush it
- For a **question mark**: write the hook then the dot as two deliberate strokes

> **Practice:** Write the sentence below, hitting every punctuation mark deliberately.
> *Hello, how are you? I am fine. It's a great day!*

---

### Problem 2 — Words appearing on a new line instead of continuing a sentence

This happens when you lift the pencil too long between words. Scribble interprets the pause as "end of line" and adds a line break.

**Fixes:**
- Keep your pencil **hovering close to the screen** between words — you don't need to fully lift away
- Write words **closer together** in a flowing motion rather than word-by-word with long pauses
- If it happens: use the **join gesture** — draw a vertical line connecting the stranded word back up to the sentence above
- Go to **Settings → Apple Pencil → Scribble** and ensure Scribble is on — the delay can't be changed, but good technique removes the problem

> **Practice:** Write this without lifting too far between words:
> *The quick brown fox jumps over the lazy dog*

---

### Problem 3 — Where to draw on the screen

Scribble doesn't require you to write in the exact spot where you want text — it processes everything written in the **active text field** (the whole note body in Obsidian) and inserts at your cursor.

**Key zones:**

```
┌─────────────────────────────────────┐
│  Obsidian toolbar (tap, don't write) │  ← Tap with pencil for buttons
├─────────────────────────────────────┤
│                                     │
│   NOTE BODY — write anywhere here   │  ← Scribble zone
│   Text inserts at cursor position   │
│   You can write large across the    │
│   whole width — it all converts     │
│                                     │
├─────────────────────────────────────┤
│  Formatting toolbar (bottom)        │  ← Tap for bold/italic/etc
└─────────────────────────────────────┘
```

**In practice:**
- Tap in the body to set cursor → write anywhere in the body area in large comfortable script
- You don't need to write small — write as large as feels natural, Scribble handles it
- The **margins** are fine to write in — Scribble sees the whole text field, not just the line

## Apple Pencil Pro Gestures (Hardware)

These work system-wide, not just in Obsidian:

| Gesture | Action |
|---|---|
| **Squeeze** (flat side) | Switches between your last two tools — customise in Settings → Apple Pencil |
| **Double-tap** (flat side) | Same as squeeze by default — configurable |
| **Hover** | Pencil tip near screen without touching shows a preview cursor — useful for precise placement |
| **Barrel roll** | Rotating the pencil changes brush angle in drawing apps — not relevant for text |

> **Tip:** Set Squeeze to "Show Colour Palette" or leave as "Switch between current tool and eraser" — in Obsidian it won't do much either way, but hover is useful for placing your cursor precisely before committing.

---

## Quick Workflow for Writing in Obsidian

1. Open note → tap pencil icon (Edit mode)
2. Tap where you want to insert text — watch for the cursor
3. Write in comfortable large print anywhere in the body
4. For punctuation — slow down, make it deliberate
5. If a word lands on the wrong line — draw the join gesture to pull it back up
6. To fix a word mid-sentence — scratch it out, then write the correction above the scratch
