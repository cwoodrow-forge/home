# 7 Days of Viral Twitter Content

**What it does:** Turns your existing long-form content (newsletter, article, YouTube script) into 7 tweets using proven structures — in about 60 minutes.

**How to use:** Gather your source content, open an AI chat, paste the prompt below, and provide your material as context.

---

## Prompt

```
You are an experienced Twitter ghostwriter who has written thousands of viral tweets. Your task is to create tweets using the ideas from my content.

For each tweet:
1. Extract a valuable insight or idea from my content
2. Format it according to one of the template structures below
3. Maintain my authentic voice and style
4. Make sure each tweet stands alone as valuable content

Create 7 different tweets, each using a different structure:
1. Contrarian take / hot take
2. Before/After transformation
3. Specific number + benefit (e.g., "3 ways to...")
4. Question that triggers curiosity
5. Actionable tip with clear outcome
6. Mistake to avoid + better alternative
7. Bold statement + supporting insight

For each tweet:
- Write it in plain text format
- Include the structure name as a label
- Ensure it's under 280 characters
- Use strategic line breaks for readability
- Optimize for engagement while maintaining authenticity

The tweets should feel like they were written by me, just formatted specifically for Twitter.
```

---

## Pro Tips

- Aim for 70% batched content, 30% spontaneous engagement
- Create a writing style document by asking AI to analyze your past tweets
- Run this process weekly to stay consistent without burning out
- Schedule the 7 tweets using a tool like Typefully
