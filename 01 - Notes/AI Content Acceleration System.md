# AI Content Acceleration System

**What it does:** A complete 3-phase workflow — idea generation, content creation, and distribution across platforms. Uses AI document referencing for context-aware output.

---

## Phase 1: Idea Generation

Open a new AI chat. Add your notes on industry trends, content philosophy, audience research, past content.

```
Based on the knowledge in my documents, generate 15 [CONTENT TYPE] ideas that would appeal to my [TARGET AUDIENCE]. For each idea, include:

1. A compelling title that would attract clicks
2. 3 key points to cover within the content
3. Why my audience would find this valuable
4. A potential hook or opening that grabs attention

Focus on topics related to [YOUR NICHE/EXPERTISE].
```

Then evaluate the top 3:

```
Review these content ideas and identify which 3 would be most likely to perform well based on current [PLATFORM] algorithm preferences and audience trends. For each of the top 3 ideas:

1. Explain why it would likely perform well
2. Suggest how to optimize it for maximum visibility
3. Identify any potential challenges in executing it
```

---

## Phase 2: Content Creation

Open a new AI chat. Add your style guide, best past content, research notes, audience feedback.

```
Using the referenced documents as context for my style, audience preferences, and expertise, create a detailed [CONTENT TYPE] outline for "[SELECTED IDEA TITLE]."

Include:
1. A powerful pattern-interrupt intro that hooks the audience
2. A clear structure with logical flow between sections
3. Three key components/points with practical examples for each
4. Anticipated viewer questions/objections and how to address them
5. A compelling call-to-action

Make sure it sounds like me and incorporates my philosophy on [YOUR KEY THEMES/PHILOSOPHIES].
```

---

## Phase 3: Distribution

### Twitter/X

```
Using my [CONTENT TYPE] as source material, create 7 different standalone tweets that extract the most valuable insights. Use these 7 structures:

1. Contrarian take / hot take
2. Before/After transformation
3. Specific number + benefit (e.g., "3 ways to...")
4. Question that triggers curiosity
5. Actionable tip with clear outcome
6. Mistake to avoid + better alternative
7. Bold statement + supporting insight

Each tweet: under 280 characters, strategic line breaks, authentic voice.
```

### LinkedIn

```
Using my [CONTENT TYPE] as source material, create 3 LinkedIn posts that drive traffic to the original content. For each post:
1. Compelling hook for professionals
2. One valuable insight with immediate value
3. Personal perspective or experience
4. Clear call-to-action

Format with spacing and bullet points for LinkedIn readability.
```

### Newsletter

```
Using my [CONTENT TYPE] as source material, create an email newsletter that:
1. Opens with a personal, conversational introduction
2. Extracts 3 key insights with immediate value
3. Includes a personal reflection or behind-the-scenes perspective
4. Ends with a compelling call-to-action

Warm, personal tone — like a message from a trusted friend.
```

---

## Principles

- **Sacred creation time** — block unassisted writing time. AI amplifies your ideas, not replaces them
- **Review everything** — treat AI outputs as first drafts that need your voice and judgment
- **AI for scaling, you for originating** — the ideas come from you, AI handles the distribution
