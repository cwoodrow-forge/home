# 04 - System — Folder Guide

This folder contains vault infrastructure: templates and attachments.

---

## What Lives Here

### Templates/
Obsidian note templates. Use these via **Templates** core plugin or **Templater** community plugin.

| Template | Use it to create |
|---|---|
| `Newsletter Template.md` | A new newsletter draft |
| `YouTube Template.md` | A new YouTube video script |
| `Weekly Tweets Template.md` | A weekly batch of social posts |

### Attachments/
All binary files: images, PDFs, exported ZIPs. When Obsidian pastes an image, it lands here automatically (configure in Settings → Files & Links → Default location for new attachments → `04 - System/Attachments`).

---

## Rules

- Never put regular notes here — they belong in `01 - Notes/`
- Templates don't need YAML frontmatter — they're scaffolding, not notes
- Configure Obsidian to save attachments to `04 - System/Attachments/`
